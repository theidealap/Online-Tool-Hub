1. Countdown To Date
========================================

This is the highest-priority issue.

Example:

https://online-tool-4u945rjrn-online-tool-hub.vercel.app/tools/countdown-to-date?target=1784223840000

Current Result:
404: NOT_FOUND
Code: NOT_FOUND

Expected Result:
The React/Vite application should load normally, restore the countdown state, and display the correct countdown.

Please investigate the root cause.

Most likely areas to review:
- Vite SPA routing
- Vercel deployment configuration
- Deep-link handling
- Refresh behavior on nested routes
- Rewrite/fallback configuration

Requirements:
- Shared countdown URLs must work.
- Direct navigation to countdown URLs must work.
- Browser refresh must work.
- Countdown state must still restore correctly.

Verify the fix on the deployed production route structure.

2. QR Code Generator
========================================

Issue:

The URL validation appears too restrictive.

Example:

https://google.com

Rejected:

Both are valid URLs and should be accepted.

Requirements:
- Review URL validation logic.
- Accept valid HTTPS and HTTP URLs.
